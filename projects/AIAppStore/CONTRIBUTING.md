
# Contributing to AIAppStore

Thank you for your interest in contributing to AIAppStore!

## How to Contribute

- **Report Bugs**: Create an issue with a clear description.
- **Suggest Enhancements**: Include detailed ideas in an issue.
- **Submit Pull Requests**: Fork the repository, create a branch, and submit your changes.

## Development Workflow

1. Clone the repository.
2. Test your changes.
3. Submit your pull request.
