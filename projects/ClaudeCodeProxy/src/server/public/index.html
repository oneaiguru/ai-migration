Getting Started

To use Claude Code with this proxy server:

  1. Get an authentication token from your administrator
  2. Download and run the client setup script
  3. Start using Claude Code as normal
# Download the client setup script
curl -O https://code.aai.guru/client-setup.sh
chmod +x client-setup.sh

# Run the setup script
./client-setup.sh -u https://code.aai.guru -t YOUR_TOKEN_HERE

# Source the configuration
source ~/.claude_proxy_config

# Test Claude Code
claude -p "Hello, Claude!"
