# Update Project Script and Test

This script updates a target project folder with the latest ZIP file from a downloads folder.

## Files Included
- `update_project_from_zip.py`: The script that performs the update.
- `test_update_project.py`: A simulated test for validating the script functionality.

## How to Use
1. Place the ZIP file in the downloads folder.
2. Run the script: `python3 update_project_from_zip.py`.

Ensure the target folder exists before running the script.
