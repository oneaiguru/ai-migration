**Scope:**
- Commit tagging (`xfeat::`, `xproto::`) per Saturday prep checklist
- Potential helper scripts to scope `git diff --stat` to relevant directories
- Documentation in `docs/SOP/saturday_prep_checklist.md` and `docs/System/workflow-metrics.md`
