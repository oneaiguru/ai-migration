# Руководство по реализации историй

## Обязательные истории

1. «Тайна библиотечных теней» (Free Tier)

    * Место: Старинная библиотека
    * Основные механики: Документальный анализ, опрос свидетелей
    * Уровень сложности: Начальный
2. «Исчезающая коллекция» (Premium Tier)

    * Место: Художественная галерея
    * Основные механики: Визуальный анализ улик, эксперты
    * Уровень сложности: Средний
3. «Дело о подделке» (Premium Tier)

    * Место: Финансовый район
    * Основные механики: Финансовые документы, видеонаблюдение
    * Уровень сложности: Продвинутый
