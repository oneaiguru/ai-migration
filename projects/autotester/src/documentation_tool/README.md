
# Documentation Tool

This tool extracts module, class, and method docstrings from Python files and generates Markdown documentation.

## Usage
- Place Python files in a directory.
- Run the tool to generate Markdown files for each Python file with docstrings.

## Example
```
documentation_tool.py <source_directory> <output_directory>
```

## Limitations
- Only processes files with valid Python syntax.
- Ignores files without docstrings.
