
# TODO Comment Aggregator

This tool detects TODO and FIXME comments in Python files and generates a summary in Markdown format.

## Usage
- Place Python files in a directory.
- Run the tool to generate a Markdown summary of TODO and FIXME comments.

## Example
```
todo_comment_aggregator/todo_comment_aggregator.py <source_directory> <output_file>
```

## Limitations
- Only detects comments marked as TODO or FIXME.
