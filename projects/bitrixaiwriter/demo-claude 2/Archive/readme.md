# Демо-система переписывания текстов с Claude API

Это минимальная демонстрационная версия системы для автоматического переписывания описаний товаров с помощью Claude API.

## Установка

1. Создайте директорию для проекта, например, `demo-claude`
2. Скопируйте все файлы в эту директорию
3. Убедитесь, что у вас установлен PHP 7.4+ с расширениями curl и json
4. Создайте поддиректории `uploads` и `results` и сделайте их доступными для записи:
   ```
   mkdir uploads results
   chmod 755 uploads results
   ```
5. Замените API ключ Claude в файле `api.php` на ваш актуальный ключ
6. Запустите проект через веб-сервер (например, PHP встроенный сервер):
   ```
   php -S localhost:8000
   ```
7. Откройте браузер и перейдите по адресу http://localhost:8000

## Структура файлов

```
demo-claude/
├── index.html         # Главная страница с интерфейсом
├── api.php            # API-обработчик запросов
├── ClaudeClient.php   # Клиент для работы с Claude API
├── PromptGenerator.php # Генератор промптов
├── sample.csv         # Образец CSV для тестирования
├── uploads/           # Директория для загруженных файлов
└── results/           # Директория для результатов обработки
```

## Использование

1. Загрузите CSV файл с описаниями товаров
   - CSV должен содержать колонку `description` с текстами для обработки
   - Дополнительно можно включить колонку `product_name` для идентификации товаров
   - Используйте `sample.csv` как образец формата

2. Просмотрите предварительный просмотр данных и нажмите "Запустить обработку"

3. Отслеживайте прогресс обработки

4. После завершения вы сможете скачать CSV файл с результатами

5. Для быстрого теста вы можете воспользоваться формой "Тестирование на одном тексте" внизу страницы

## Примечания

- Скорость обработки зависит от количества записей и скорости ответов Claude API
- API ключ имеет ограничения на количество запросов, учитывайте это при обработке больших файлов
- Система не подключается к реальному API Битрикс, это только демонстрация
