# Демо-система переписывания текстов с Claude API

Это минимальная демонстрационная версия системы для автоматического переписывания описаний товаров с помощью Claude API.

## Установка и запуск

1. Создайте директорию для проекта, например, `simple-claude-demo`
2. Скопируйте все файлы в эту директорию:
   - `index.html` - основной интерфейс
   - `api.php` - серверный API-обработчик
   - `ClaudeClient.php` - клиент для работы с Claude API
   - `PromptGenerator.php` - генератор промптов

3. Убедитесь, что у вас установлен PHP 7.4+ с расширениями curl и json

4. Замените API ключ Claude в файле `api.php` на ваш актуальный ключ

5. Запустите проект через веб-сервер (например, PHP встроенный сервер):
   ```
   php -S localhost:8000
   ```

6. Откройте браузер и перейдите по адресу http://localhost:8000

## Использование

1. Вставьте CSV данные в текстовое поле
   - Данные должны содержать колонки `ID`, `Название` и `Описание`
   - Разделитель должен быть `;` (точка с запятой)
   - Ограничение: до 30 строк (для производительности браузера)

2. Нажмите "Распознать данные" для предварительного просмотра

3. Нажмите "Запустить обработку" для генерации новых описаний

4. После завершения вы увидите результаты и сможете просматривать сгенерированные описания

5. Для быстрого теста вы можете воспользоваться формой "Тестирование на одном тексте" внизу страницы

## Примечания

- Скорость обработки зависит от количества записей и скорости ответов Claude API
- API ключ имеет ограничения на количество запросов, учитывайте это при обработке файлов
- Система не подключается к реальному API Битрикс, это только демонстрация
