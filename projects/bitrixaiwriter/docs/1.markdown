1. Откуда взять товары для обработки
1. 

Теперь система работает с URL товаров вместо ID, как вы и просили:

1. **Создайте Excel или CSV файл** с URL товаров в первой колонке
2. **Загрузите файл** через веб-интерфейс (перетащите или выберите)
3. Система **автоматически извлечет данные** по каждому URL
4. Вам **не нужно знать ID** - достаточно ссылок на товары

2. Два режима работы (как вы предложили)

**Фаза 1 (сейчас готово):**

- Вход: CSV/Excel с URL товаров
- Обработка: генерация новых описаний
- Выход: CSV файл с результатами
- Без прямой интеграции с сайтом

**Фаза 2 (до 21 мая):**

- Полная интеграция через API
- Автоматическое обновление на сайте
- Массовая обработка по фильтрам

3. Где просмотреть результаты и ошибки

Добавил удобный интерфейс для работы с результатами:

- **Фильтрация по статусу** (успешно, ошибки, низкая уникальность)
- **Детальный просмотр** каждого результата
- **Экспорт в CSV** для дальнейшей работы
- **История всех обработок** с возможностью скачивания

4. Кнопка "Приостановить"

Кнопка активируется только во время обработки. Когда процесс не запущен, она неактивна - это нормальное поведение.

5. Редактор промптов

Добавил полноценный редактор промптов в раздел "Настройки":

- **Визуальный редактор** с подсветкой синтаксиса
- **Готовые шаблоны** для разных типов товаров
- **Предпросмотр результата** на тестовом товаре
- **Сохранение версий** промптов
