# Пошаговая демонстрация работы системы автоматизации текстов

## Начало работы с системой

Давайте пройдем каждый шаг работы с системой и ответим на ваши вопросы.

### 1. Доступ к товарам для обработки

**Вопрос: Откуда взять товары на запуск обработки или откуда и как их подгрузить?**

Система автоматически получает товары из вашего сайта на Битриксе через API. Вам не нужно делать отдельную выгрузку или загрузку. 

#### Как это работает:

1. Когда вы запускаете обработку, система делает API-запрос к вашему сайту
2. Система получает товары согласно указанным фильтрам
3. Обработка происходит автоматически

**Чтобы запустить обработку, заполните следующие поля:**



- **Количество товаров**: Введите число товаров для обработки (например, 100)
- **Фильтр (JSON)**: Этот параметр позволяет выбрать конкретные товары по разделу, бренду и т.д.

**Примеры фильтров:**
- Для обработки товаров из конкретного раздела:
  ```json
  {"SECTION_ID": 10}
  ```
- Для обработки товаров определенного бренда:
  ```json
  {"PROPERTY_BRAND": "АКТАКОМ"}
  ```

Если оставить поле фильтра пустым, система будет обрабатывать все товары последовательно.

### 2. Идентификация доступных товаров

**Вопрос: Какие товары подгружены, чтобы ввести id?**

Для тестирования на одном товаре вам нужно знать его ID. Вот способы получить ID товаров:

#### Способ 1: Через админку Битрикс
1. Откройте админку вашего сайта
2. Перейдите в раздел "Каталог" → "Товары"
3. Найдите нужный товар, его ID обычно отображается в таблице или при открытии карточки товара

#### Способ 2: Через отчет экспорта
1. В нашей системе перейдите во вкладку "Экспорт"
2. Запустите экспорт с небольшим лимитом (10-20 товаров)
3. Система покажет список товаров с их ID

#### Способ 3: Через тестовый запрос API
Если у вас есть доступ к API Битрикс, можно сделать тестовый запрос:
```
GET /rest/catalog.product.list.json?limit=10
```

### 3. Просмотр ошибок обработки

**Вопрос: Где можно просмотреть список товаров с неудачной обработкой?**

Для просмотра товаров с ошибками:

1. Перейдите в раздел "Управление текстами" в левом меню
2. Используйте фильтр "Статус" и выберите значение "Ошибка"
3. Нажмите кнопку "Применить фильтр"



Система покажет все товары, которые не удалось обработать корректно. Для каждого товара вы можете:
- Просмотреть детали ошибки
- Повторить обработку
- Отредактировать текст вручную

### 4. Управление процессом обработки

**Вопрос: Кнопка приостановить не активна**

Кнопка "Приостановить" активируется только когда есть запущенный процесс обработки. Если процесс еще не был запущен или уже завершился, кнопка будет неактивной.

**Для запуска процесса:**
1. Заполните параметры обработки (количество товаров, фильтры)
2. Нажмите "Запустить обработку"
3. После запуска кнопка "Приостановить" станет активной

### 5. Настройка промптов

**Вопрос: В настройках я не нашла редактор промптов**

Редактор промптов находится в расширенных настройках:

1. Перейдите в раздел "Настройки" в левом меню
2. Прокрутите вниз до блока "Параметры генерации"
3. Нажмите на кнопку "Расширенные настройки" или "Редактировать промпт"

## Дополнительные настройки системы

### Настройка параметров генерации

Для оптимизации генерации текстов вы можете настроить следующие параметры:

1. **Температура**: Контролирует креативность текста
   - Низкая (0.1-0.3): Более консервативные тексты, ближе к оригиналу
   - Средняя (0.4-0.7): Баланс между уникальностью и сохранением структуры
   - Высокая (0.8-1.0): Максимальная творческая свобода

2. **Максимальное количество токенов**: Ограничивает длину генерируемого текста
   - 500-700: Для коротких описаний
   - 1000-1500: Для стандартных описаний
   - 2000+: Для подробных описаний

### Автоматический запуск обработки по расписанию

Система позволяет настроить автоматический запуск обработки по расписанию:

1. Перейдите в раздел "Настройки" → "Расписание"
2. Настройте периодичность запуска (ежедневно, еженедельно)
3. Укажите параметры запуска (количество товаров, фильтры)
4. Нажмите "Сохранить расписание"

Это позволит системе автоматически обрабатывать товары без вашего участия.
