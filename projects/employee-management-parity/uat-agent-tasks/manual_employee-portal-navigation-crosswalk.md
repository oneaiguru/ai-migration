## Как использовать
1. Откройте указанные главы в `${MANUALS_ROOT}/estimation/processing_manual/process/chapters/` до начала UAT; храните изображения (рис.) для сравнения.
2. Во время прохода демо повторяйте реальные шаги из Naumen, фиксируйте несовпадения прямо в `docs/Tasks/uat-packs/parity_static.md` и `trimmed_smoke.md` с ссылкой на раздел manual.
3. Делайте скриншоты по алиасам из `docs/SCREENSHOT_INDEX.md` (например, `portal-requests-playwright.png`) — соотнесите с manual (рис.60.x).
4. Любые расхождения, которые остаются вне текущего объема (например, обмен сменами), документируйте в Open Gaps CodeMap + UAT Findings.
