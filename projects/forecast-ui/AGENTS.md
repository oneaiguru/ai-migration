# Forecast UI
- What: Frontend UI for the RTNEO forecasting experience (components, hooks, docs, e2e assets) built with Vite/React/TS.
- Install: `npm ci`
- UI build: `npm run build`
- Unit tests: `npm run test`
- E2E/tests: `npm run test:e2e:serial` (set `E2E_BASE_URL` as needed); prod check: `npm run test:e2e:prod`
- Dev server: `npm run dev`

Monorepo note: /Users/m/ai is a multi-project monorepo. See the root README for details.
