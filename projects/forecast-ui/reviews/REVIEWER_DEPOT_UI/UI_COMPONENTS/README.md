# UI Components Bundle

Includes key React components used by reviewers to check UI behavior and structure.

Files
- code/Layout.tsx — main layout with sidebar + tabs
- code/Sidebar.tsx — navigation menu
- code/Overview.tsx — metrics cards and charts
- code/Districts.tsx — districts summary + pie
- code/Sites.tsx — sites screen with filters and paging
- code/Routes.tsx — routes screen (cards/table switch)
- code/RoutesTable.tsx — routes table wrapper (paging, CSV)
- code/PlanAssignments.tsx — badges + critical filter
- code/RegistryView.tsx — registry listing view
- code/shared/RiskBadge.tsx — shared badge
- code/shared/FillBar.tsx — shared progress bar
