# Сверка факта вывоза

### Суммарный объем за день

> **/api/v1/lk/getVolumeReportGroupedByDay**

Возвращает суммарные объемы двух видов, запланированные и фактические, по всем КП которые вывозились в указанный день.

| PARAMS | value | **Комментарий** |
|--------|-------|-----------------|
| dateFrom | 2024-05-14T10:00:00Z | Дата с : format yyyy-mm-ddThh:mm:ssZ |
| dateTo | 2024-05-15T10:00:00Z | Дата до : format yyyy-mm-ddThh:mm:ssZ |

Пример запроса:\
curl --location --request POST

`https://disp.t1.groupstp.ru/app/api/v1/lk/getVolumeReportGroupedByDay?dateFrom=2024-05-20&dateTo=2024-05-21`

<details>
<summary>Результат</summary>

```json
{
    "id": "e095b8bf-3db9-4b44-b7b3-f1d066747276",  // Рандомный id отчета (можно не использовать, на случай если нужно определить, что ответ незакеширован)volumes - объемы сгруппированные по дням
    "dateFrom": "2024-05-20",                      // Дата от
    "dateTo": "2024-05-21",                        // Дата до


    "volumes": [                                   
        {
            "date": "2024-05-20",                  // Дата

        },
        {
            "date": "2024-05-21",                  // Дата
            "plan": 23168.049999999945,            // Запланированный объем
            "fact": 19758.750000000073             // Фактический объем
        }
    ]
}
```

</details>

### Объем по каждой КП за день

> **/api/v1/lk/getVolumeReportGroupedByArea**

Возвращает объемы двух видов, запланированный и фактический, по всем КП реестра и указанной дате.

| PARAMS | value | **Комментарий** |
|--------|-------|-----------------|
| date | 2024-05-14T10:00:00Z | Дата : format yyyy-mm-ddThh:mm:ssZ |

Пример запроса:\
curl --location --request POST

`https://disp.t1.groupstp.ru/app/api/v1/lk/getVolumeReportGroupedByArea?date=2024-05-20`

<details>
<summary>Результат</summary>

```json
{
    "id": "7d412d32-bd66-4bcf-be4f-54014d290e6f",  // Рандомный id отчета (можно не использовать, на случай если нужно определить, что ответ не закеширован)
    "date": "2024-05-20",                          // Дата (параметр запроса)
    "volumes": [                                   // Объемы, сгруппированные по КП
        {
            "areaLkCode": "38013987",              // Код КП
            "plan": 8.0,                           // Запланированный объем
            "fact": 8.0                            // Фактический объем
        },
        {
            "areaLkCode": "38100000",              // Код КП
            "plan": 1.5,                           // Запланированный объем
            "fact": 0.75                           // Фактический объем
        }
     ]
}
```

</details>
