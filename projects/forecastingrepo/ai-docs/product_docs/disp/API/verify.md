# Верификация

## Каfka

### Отправка верифицированных КП

Отправка верифицированных КП происходит, когда КП полностью согласована (на фронте будет отмечено РЕШЕНО и СОГЛАСОВАНО).

Все ответы приходят в комнату **VERIFICATION_LK_IN\_{server_name}**, где server_name - суффикс сервера.

```json
{
    "verificationId": "77db34b5-56f9-4cb0-87e9-eb08df333aab",         // ID Верификации
```
