# Полигоны

Раздел предназначен для аудита весовых данных, получаемых с полигонов и МПС в виде Excel-файлов, и сверки их с весовыми данными от водителей. Раздел доступен пользователям с ролью "регоператор".

## Настройка

Для корректного импорта данных для каждого места разгрузки нужно описать формат Excel-таблицы, в котором представлены данные с него. Выберите нужное место из списка и укажите значения, стоящие в столбцах. Например формат таблицы вида -

![](2024-11-10-19-41-40.png)

описывается как -

![](2024-11-10-19-42-47.png)

Для импорта данных используются только дата и время загрузки, номер ТС и вес. Остальные столбцы в импорте не участвуют, поэтому в описании формата они пропускаются.

## Импорт данных

Выберите нужное место разгрузки и импортируйте Excel-файл для него. Если в Excel-файле несколько листов, будет импортирован только первый.

Строка из файла не будет импортирована, если -

1. Данные в ней не соответствуют описанному формату.
2. Номер ТС отсутствует в справочнике ТС.

Соотв. сообщение об ошибке будет показано после импорта данных.

## Сверка по весу

Отчет показывает суммарные данные за выбранный интервал времени по всем местам разгрузки как по данныи полигонов, так и по данным из ПО, с возможностью сравнить их. Данные с полигонов (мест захоронений ТКО) и МСП (мест перегрузки ТКО) сравниваются отдельно.

Данные по каждому полигону можно развернуть и увидеть их детализацию по перевозчкам, дням или ТС. Для этого нужно, например, в столбце "Перевозчик" нажать на ссылку "Все" -

![](2024-11-10-20-56-30.png)

Аналогично можно раскрыть детализацию по дням и по ТС. Если каких-то данных нет, в соотв. ячейке будет стоять прочерк.

![](2024-11-10-20-58-38.png)

Столбец "Дата последней поверки" показывает дату, когда на объекте в последний раз проводилась поверка весов. Дата указывается в разделе Настройка - Гео-объекты - Весы для каждого объекта.

## Сверка по местам разгрузки

Отчет помогает найти расхождения в количестве весовых отчетов и местах разгрузки с каждой ТС по данным полигонов и данным в ПО.
