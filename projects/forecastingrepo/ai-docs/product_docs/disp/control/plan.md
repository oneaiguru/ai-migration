# Планирование баланса масс

Раздел позволяет регоператору задать генеральным перевозчикам план распределения ТКО по весовым объектам (полигонам и МПС). Генеральные перевозчики, в свою очередь, разбивают этот план между своими субподрядчиками.

### Настройка

Перед началом планирования в Общих настройках нужно настроить параметр "Планирование объектов разгрузки":

![](2024-07-26-11-09-41.png)

- Тип планирования - только 2е плечо или 1+2 плечо вместе
- Интервал планирования - на день или на месяц
- Единица измерения плана - тонны или проценты

У генеральных перевозчиков в Настройке - Организации должна быть включена опция "Генеральный перевозчик".

Для корректного распределения планов между субподрядчиками у каждой машины в справочнике ТС должно быть указано поле "Генеральный перевозчик".

Пользователям, имеющим право создавать и редактировать планы, в справочнике сотрудников добавляется роль "Планирование баланса масс".

### Планирование регоператором

В разделе Контроль - Объекты нужно выбрать дату планирования и заполнить планы для генеральных перевозчиков на эту дату.

#### 1е плечо

При планировании 1го плеча нужно указать, сколько тонн/процентов нужно перевезти с выбранного участка на выбранный объект разгрузки генеральному перевозчику и его субподрядчикам. Регоператор не делает детальную разбивку плана по субподрядчикам, он указывает только суммарную цифру для исполнения генеральным перевозчиком.

План указывается пользователем, факт считается Чистой логистикой на основе весовых отчетов ТС за указанную дату.

![](2024-07-26-11-40-24.png)

#### 2е плечо

При планировании 2го плеча нужно указать, сколько тонн/процентов нужно перевезти с выбранных МПС на выбранные полигоны генеральному перевозчику и его субподрядчикам.

![](2024-07-26-11-41-47.png)

Чтобы упростить планирование, показываются -

- Средний вес, перевозимый за один рейс по указанному маршруту. Эта цифра считается на основе весовых отчетов ТС 2го плеча, перевозивших ТКО по данному маршруту за прошлый месяц.

- Рекомендованное число рейсов - оценка числа рейсов, нужных для перевозки запланированного объема ТКО с учетом среднего веса одного рейса.

### Планирование генеральным перевозчиком

Пользователь генерального перевозчика по этому же адресу Контроль - Объекты видит план, поставленный ему регоператором, и список своих субподрядчиков.

![](2024-07-26-12-08-05.png)

Он должен распределить задание от регоператора между своими машинами и машинами субподрядчиков.

![](2024-07-26-12-11-12.png)

### Планирование перевозчиком

Пользователи субподрядчиков вядят план, созданный для них генеральным перевозчиком.

![](2024-07-26-12-17-14.png)

### Контроль выполнения

Для контроля выполнения плана используется столбец Факт - он показывает данные весовых отчетов водителей за указанные дату и маршрут.
