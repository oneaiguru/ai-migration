# Невывоз

Отчет по невывозу:

- отчет по невывозу содержит КП, которые были невывезены.

Отчет по невывозу содержит следующую информацию:

- Код КП
- Адрес КП
- Тариф
- График вывоза
- Собственник КП
- Госномер ТС, которое вывозило КП
- Перевозчик, который вывозил КП
- Количество невывезенных контейнеров
- Уведомление - активная кнопка для формирования уведомления собственнику КП

Отчет можно выгрузить в Excel, PDF.
