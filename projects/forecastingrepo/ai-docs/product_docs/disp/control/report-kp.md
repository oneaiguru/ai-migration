# Отчеты по КП

Отчеты по КП:

- КП без план-задания - кп, у которой график вывоза совпадает с датой формирования отчета, при этом кп нет в план-задании.
- КП без отчета - кп, которая есть в план-задании, но по ней нет отчета.
- КП без маршрута - кп, которой нет в маршруте.
- Пустые КП - КП, у которых факт вывоза указан, как пусто.
- КП без вывоза более 1 месяца - кп, которая не вывозилась 30 дней с даты формирования отчета.
- КП без контейнеров - КП, у которых в системе нет ни одного контейнера.
- Заполненность КП - Для каждой КП заранее рассчитывается 12 коэффициентов ее заполненности (по месяцам). Берем отчеты водителя за последние 365 дней, и за каждый месяц считаем -  
E = объем пустых контейнеров  
X = объем подбора  
S = общий объем вывезенных контейнеров (за вычетом невывоза)

Коэфф. заполненности за выбранный месяц будет равен (S + X - E) / S. Если по КП было получено менее 10 отчетов или на КП нет контейнеров (КГО или мешки), коэффициент считается равным 100%.
В разделе Контроль показываются значения заполненности за прошлый месяц. Если коэффициент = 100%, КП в отчете не показывается. Таблица сортируется по возрастанию коэффициента.

При экспорте в Excel показываются значения всех 12-ти коэффициентов, рассчитанных для КП. В экспорт добавляются все КП из реестра, включая те у кого коэффициент = 100%.

Каждый отчет можно выгрузить в excel, pdf.
