# Уведомления собственникам КП

Модуль предназначен для отправки информации о проблемах на КП по электронной почте собственникам КП и иным лицам.  
Модуль можно подключить в общих настройках системы. Параметр "Показывать раздел - Акты(отправка уведомлений)

Создать уведомление можно через разделы: 

- Мониторинг - [в отчете по вывозу КП](../monitoring/notice.md) 
- Контроль - [в отчете по Невывозу](../control/report-breakdown.md) 
- Контроль - [в отчете по Проблемам](../control/report-problem.md)
- Контроль - [Акты](../control/noticeAct.md)

В отчете доступна кнопка "Отправить уведомление". По нажатию на кнопку открывается окно для редактирования и отправки уведомления. 

![](noticeAct3.png)

Для отправки уведомления необходимо выполнить 2 шага:
1. Подготовка текста уведомления
2. Выбор адресатов для отправки уведомления
Для отправки уведомления необходимо выполнить 2 шага:
1. Подготовка текста уведомления
2. Выбор адресатов для отправки уведомления

#### Подготовка текста уведомления

В окне работы с уведомлением требуется подготовить текст письма или выбрать готовый шаблон. Для создания, редактирования текста письма предлагается стандартный набор инструментов.

![](noticedAct2.png)

Подготовленный текст уведомления необходимо сохранить перед отправкой. После сохранения становится доступным отправка уведомления по электронной почте или скачивание в формате PDF. Сохраненый акт попадает в раздел Контроль - Акты.

#### Выбор адресатов для отправки уведомления

Уведомление можно отправить:
- всем или нескольким собственникам КП, отметив каждого собственника галочкой - меню с выбором собственников отображается только, когда они есть в системе(поступают из ЛК), уведомления формируются на отдельных вкладках для каждого собственника КП.  
- иному адресату, указав адрес электронной почты вручную.
- скачать уведомление в PDF для отправки по иным каналам связи.

![](noticeAct6.png)

##### Создание шаблона для отправки уведомлений

Можно воспользоваться редактором шаблонов уведомлений и подготовить шаблон письма на любой случай. При отправке новых уведомлений останется только выбрать нужный шаблон и собственника. 

В списке выбора шаблонов нажмите "Редактировать шаблоны" для перехода в редактор.  

![](noticeAct5.png)

1. Для создания нового шаблона нажмите "Добавить шаблон".
2. Подготовьте текст уведомления вручную или загрузите файл word.
3. Вставьте в текст письма необходимые макросы: кому, email, ИНН, Уважаемый, Исходящий номер, район КП, адрес КП.
4. Сохраните изменения.

![](noticeAct4.png)

Клик по макросу из списка переносит его в текст. 

Все отправленные уведомления хранятся в разделе "Контроль", подраздел [Акты.](../control/noticeAct.md)

По нажатию на кнопку "Редактировать шаблон письма" открывается окно для редактирования соответствующего шаблона. 

![](noticeAct9.png)

#### Собственники КП и Контакт КП

Информация о собственниках КП поступает из Личного кабинета MyTKO.
В Чистой логистике у каждой КП есть поле "Контакт КП", которое содержит информацию:  ФИО, должность, ИНН и email. 


При создании уведомлений, если Контакт КП заполнен в Чистой логистике, то он отображается в полях ФИО, ИНН, email. Если данных о контакте КП нет, то можно использовать информацию о собственниках КП из Личного кабинета. 

У каждого собственника есть меню с одним пунктом - Использовать, как основной контакт. При его выборе данные собственника копируются в Контакт КП.
