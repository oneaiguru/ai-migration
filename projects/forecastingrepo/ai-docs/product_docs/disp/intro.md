# Чистая логистика

Чистая логистика предоставляет все необходимые инструменты для планирования и контроля вывоза отходов для регионального оператора и перевозчиков ТКО.

<iframe src="https://player.vimeo.com/video/1000058604" width="960" height="540" frameborder="0" allow="autoplay; fullscreen" allowfullscreen></iframe>

Чистую логистику используют разные пользователи:

- Водители отправляют отчеты о вывозе и весовом контроле из [мобильного приложения](prepare/mobile)
- Мастера участков следят за [отчетами водителей](monitoring/view), вносят коррективы в [маршруты](routes)
- Диспетчеры готовят [план-задания](plan/plans) на вывоз

Если эти или другие термины вызывают вопросы, прочтите [Терминологию](../prepare/terms), используемую в приложении.
