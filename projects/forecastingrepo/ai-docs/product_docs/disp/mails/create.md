# Создание писем

## По обратной связи

Письма раздела "По обратной связи" создаются на основе отчетов водителей. Есть три варианта создания такого письма:

### 1. Мониторинг

Откройте отчет водителя в мониторинге и нажмите на ссылку "Отправить письмо..."

![](2024-04-29-16-59-05.png)

### 2. Отчеты о невывозе и проблемах КП

Откройте отчет водителя в невывозе или проблемах КП и выберите в меню команду "Отправить письмо..."

![](2024-04-29-17-01-14.png)

### 3. Раздел "Письма"

Нажмите кнопку "Создать письмо" в разделе "Письма - По обратной связи", укажите КП и выберите отчет водителя из списка десяти последних отчетов.

![](2024-04-29-17-02-12.png)

### Создание текста письма

#### 1. Выберите шаблон с подходящим содержанием

![](2024-04-29-17-09-35.png)

#### 2. Укажите контакты контрагента

Если в реестре КП у площадки был указан [контакт для уведомлений](../registry/contacts), его данные будут подставлены в форму автоматичесски. Если нет, введите данные вручную.

![](2024-04-29-17-17-29.png)

#### 3. Проверьте содержание письма

В письме не должно остаться нераскрытых [макросов](template). Отредактируйте текст при необходимости и поставьте письму статус "Готов к отправке". 

![](2024-04-29-17-20-23.png) 

## Отказ верификации

Выберите заявку на верификацию, по которой есть заполненный акт, и в меню выберите пункт "Отправить письмо..."

![](2024-04-29-17-05-07.png)

## Перевозчикам и Снятие КП

Нажмите кнопку "Создать письмо" в соответствующем разделе.
