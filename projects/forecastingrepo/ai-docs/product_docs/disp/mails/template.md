# Шаблоны писем

Для редактирования шаблонов выберите "Редактировать шаблоны..." в списке шаблонов.

![](2024-04-29-17-39-46.png)

Нажмите "Добавить шаблон", укажите его название, а затем загрузите заготовку из Word-файла.

![](2024-04-29-17-40-45.png)

Изменяемые части в тексте замените на макросы. Их значения будут подставлены при создании нового письма из данного шаблона. При этом макросы "КОМУ" и "УВАЖАЕМЫЙ" автоматически поставят ФИО контакта в нужный падеж и род.

![](2024-04-29-17-42-26.png)
