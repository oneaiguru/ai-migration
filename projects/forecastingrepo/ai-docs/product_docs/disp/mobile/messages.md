# Сообщения водителям

Администратор может отправлять текстовые сообщения в приложение выбранных водителей или всем водителям выбранных перевозчиков. 

![](2024-04-30-18-05-16.png)

### Сообщение выбранным водителям

1. В разделе Справочники - Устройства отметьте галочкой устройства, на которые нужно отправить сообщение.
2. В меню "Управление" выберите "Отправить сообщение".
3. В открывшемся диалоге выберите "Выбранным водителям".
4. Если сообщение срочное, выберите "Немедленно", если нет - "При входе в приложение".
5. Введите текст сообщения и подпись под ним.

### Сообщение всем водителям выбранных перевозчиков

1. В разделе Справочники - Устройства в фильтре перевозчиков выберит тех, кому нужно отправить сообщение.
2. В меню "Управление" выберите "Отправить сообщение".
3. В открывшемся диалоге выберите "Всем (с учетом фильтра)".
4. Если сообщение срочное, выберите "Немедленно", если нет - "При входе в приложение".
5. Введите текст сообщения и подпись под ним.
