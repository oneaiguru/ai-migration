# Обновление

## Обновление от 27.03.2025

### Показывать предупреждение об устаревшем ПЗ

Если дата ранее сохранённого ПЗ не соответствует выбранной, будет отображаться предупреждение о том, что используемое ПЗ устарело.

Для ТС первого плеча появляется сообщение:

Для ТС второго плеча появляется сообщение:

![](<image (95).png>)

## Обновление от 16.07.2024

### Предупреждение о повторном невывозе

Если на КП из план-задания водителя ранее был зафиксирован невывоз, водителю будет показано предупреждение об этом, чтобы не допустить повторного невывоза.

![](2024-07-16-12-20-26.png)

### Минимальное время между первым и вторым рейсом

В настройках приложения можно задать минимальное время между первым и вторым рейсом. После этого приложение будет блокировать отчет с КП, если водитель ранее уже вывозил ее и указанное время еще не прошло.

![](2024-07-16-12-22-56.png)

### Навигатор

В приложении появился полноценный режим навигатора. Чтобы перейти в него, надо нажать кнопку "Навигация" в окне КП -

![](2024-07-16-12-29-46.png)

Чтобы выйти из навигатора, достаточно переключиться в режим прокладки маршрута на карте или обратно в план-задание.

![](2024-07-16-12-31-43.png)

### Запрет подбора на КП

Регоператор может запретить подбор ТКО и КГО на некоторых КП. На таких КП ввод объема подбора будет заблокирован.

## Обновление от 15.06.2023
#### Улучшения и оптимизация:
- теперь в списке заявок на Верификацию отображаются только не согласованные заявки.

Когда приложение находится в режиме верификации КП мастером, то теперь мастер видит только не согласованные заявки, а согласованные заявки мастеру не показываются.

- для избежания ошибки с выбором не правильного план-задания, теперь водитель может выбрать ПЗ на завтра только после 12.00 текущего дня. 

## Обновление от 23.01.2023
#### Новые функции:
- прицеп для второго плеча

В приложении у ТС 2го плеча добавляется переключатель "Прицеп". Отдельно для каждого отчета можно указать вес прицепа с ТКО и без.

![](2301-1.jpg)

#### Улучшения и оптимизация:
- если указана причина невывоза "Не вошло в рейс", "Поломка автомобиля", то фото водитель может не делать
- указание мест загрузки и выгрузки для ТС 2го плеча

Для 2го плеча на вкладке Вес приложение само выбирает показывать список мест разгрузок или загрузок, которые были определены для ТС в редакторе план-заданий.  
Водитель может выбирать из предложенного списка нужный объект либо выбрать на карте любой другой.

## Обновление от 29.11.2022

#### Новые функции:

- в приложении водителя теперь отображаются пробки на карте 2Гис

![](release2911-1.jpg)

По умолчанию пробки включены, их можно отключить нажатием на иконку внизу слева (она показывает значение пробок в баллах от 0 до 10). Иконки может не быть в деревнях, за городом и в других подобных ситуациях, когда 2Гис не строит пробки по данной территории.

## Обновление от 10.11.2022
#### Новые функции:

- несколько смен у одной машины 

На одном ТС теперь может работать несколько экипажей в течение одного дня - в две и реже в три смены. Маршрут готовится отдельно для каждого экипажа. План-задание также создается отдельно для каждого экипажа.

В приложении у ТС, где больше одного экипажа, заголовок "За рулем" меняется на выбор номера смены -

![](double-crew8.png)

Приложение получает план-задание для этой смены, а также отправляет отчеты от ее имени, а не просто от номера ТС.[см. подробнее](../plan/double-crew.md)

#### Улучшения и оптимизация:
- добавлен скролл для пролистывания объемных комментариев
- добавлены два новых типа мест разгрузки 
1. МПС + МСС. Означает перегруз, комбинированный с сортировкой
2. КПО. Комплекс переработки отходов, или мусороперерабатывающий завод

- название объектов при разгрузке\загрузке на вкладке Вес указываются явно. Ранее указывались только номера рейсов
- теперь водитель может отправить пустой отчет с КГО-площадки. 

#### Исправления ошибок:
- ошибки с отправкой отчетов о вывозе и фотографий из-за сбоев со связью
