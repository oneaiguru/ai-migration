# Содержание
                   
Мониторинг показывает план-задания на выбранную дату и ход их выполнения. 

Номера машин показываются -

- черно-белыми, если у машины есть план-задание
- бело-серыми, если это мусоровоз второго плеча
- приглушенными, если план-задание не содержит КП

![](icons.png)

В план-задании каждая КП имеет один из 3 статусов:
- серая иконка - КП без отчета
- зеленая иконка - КП с отчетом
- красная иконка - КП с отчетом, но есть невывоз

![](intro3.png)

Иконки план-задания - 
- кружок - стандартное обозначение КП в ПЗ
- закрашенный кружок - КП с отчетом принято мастером
- знак плюс - перенос КП из другого ПЗ

После принятия ПЗ мастером кружки закрашиваются.

![](intro2.png)

## Разделы отчетов

![](menu.png)

- План-задание: ход выполнения план-задания водителя
- Вес: данные весового контроля
- Отчеты: отчеты водителя с фотофиксацией вывоза
