# Перенос КП между ПЗ 

Перенос может потребоваться, если водитель не может выполнить ПЗ. 
В этом случае невывезенные КП необходимо разнести в другие ПЗ.
 
Для выбора контейнерной группы кликните по чек-боксу слева от адреса и нажмите "Перенести". 
Чтобы выделить несколько КП, кликайте по чек-боксам с зажатой клавишей Ctrl или Shift. 

![](move-monitor2.png)

В окне переноса выберите необходимый участок и ТС. 
Укажите дату на которую будет выполнен перенос выбранных КП.

![](move-monitor1.png)

Можно также перенести КП мышкой, бросив их на госномер нужного ТС.
Такой перенос работает только на сегодняшнюю дату.

![](2021-10-17-16-30-29.png)
