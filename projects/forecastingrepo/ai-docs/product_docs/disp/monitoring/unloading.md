---
title: Экспорт план-заданий
---

В меню мониторинга выберите План-задание или Маршрутный журнал для экспорта план-заданий в одном из двух форматов.

![](2024-04-30-17-50-58.png)

Формат "маршрутный журнал" экспортирует данные с фактом вывоза КП. 
Формат "план-задание" - только список КП в план-задании без факта вывоза. Выгрузка сформируется для всех ТС выбранного перевозчика и участка.
