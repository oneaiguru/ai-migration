# Просмотр отчетов
                   
Нажмите на строку КП в план-задании, чтобы увидеть отчет по ней.

![](2021-10-17-16-44-12.png)

**Факт** - число забранных контейнеров, **Подбор** - объем подбора в кубометрах, **Пусто** - число изначально пустых контейнеров.

![](2021-10-17-16-45-21.png)

Факт, Подбор и Пусто можно редактировать.

Переключитесь на вкладку **Отчеты**, чтобы увидеть ленту отчетов водителя за выбранную дату.

![](2021-10-17-16-49-04.png)
