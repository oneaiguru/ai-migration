# Добавление площадок с КГО
                   
Если в реестре КП задан график вывоза КГО с площадки, в редакторе маршрутов КГО будет представлено отдельной строкой. Его можно поместить в нужный маршрут, и в план-задание оно попадет автоматически.

![](2021-10-17-13-37-59.png)

Если нужно добавить заявку на вывоз КГО с любой площадки, нужно выбрать ее в редакторе план-заданий и нажать кнопку "Добавить КГО".

![](2021-10-17-13-43-52.png)
