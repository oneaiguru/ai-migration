# Создание ПЗ из нескольких маршрутов.

План-задание можно создать из нескольких маршрутов.
Для соединения маршрутов кликните в поле маршрут и 
перетяните ползунок "соединять маршруты" направо.
                                                     
![](merge-routes1.png)

Выберите маршруты для соединения. Выбранные маршруты сразу добавятся в текущее ПЗ.
![](merge-routes2.png)
