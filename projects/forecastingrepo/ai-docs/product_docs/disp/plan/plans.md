# Создание план-заданий
                   
Чтобы создать план-задание, в редакторе план-заданий выберите дату и ТС. Если у ТС есть маршрут по умолчанию, план-задание будет автоматически сделано на его основе.

![](2021-10-13-21-32-28.png)

План-задание останется пустым, если:

- у ТС нет маршрута по умолчанию
- у ТС задан график работы и этот день у нее выходной

Место разгрузки по умолчанию наследуется от маршрута, но может быть изменено у план-задания.

Маршрут можно изменить, выбрать любой из выпадающего списка, в этом случае план-задание будет очищено и создано заново на основе выбранного маршрута.

### Добавление и удаление КП

Чтобы добавить КП в план-задание, выберите ее в нижней таблице и нажмите "Добавить в план-задание". Используйте поиск по адресу или коду КП в области фильтров, чтобы найти КП.

![](2021-10-13-21-40-38.png)

Чтобы удалить КП из план-задания, выберите его в верхней таблице и нажмите "Удалить".

![](2021-10-13-21-41-47.png)
