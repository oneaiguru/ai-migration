# Перенос план-задания между машинами

В редакторе план-заданий можно полностью перенести ПЗ на новое ТС.  
Нажмите на иконку для переноса ПЗ. 

![](1005-2.jpg)

Выберите в модальном окне новое ТС и нажмите "выбрать". 

![](1005-1.jpg) 
 
ПЗ переместится на новое ТС.

![](1005-3.jpg)
