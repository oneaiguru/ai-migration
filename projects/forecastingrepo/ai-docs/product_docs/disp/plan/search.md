# Поиск

В редакторе план-заданий поиск разделен на два вида. 
Глобальный поиск и поиск по участку через меню фильтров. 

В редакторе план-заданий две таблицы - с текущим план-заданием 
транспортного средства и контейнерными площадками выбранного участка. 
В окне глобального поиска можно выбрать, где он будет происходить.
 
### Глобальный поиск по ПЗ покажет и ПЗ и участок с КП, которую искали.

![](search1.png)

### Глобальный поиск по участку покажет только участок с КП, которую искали.
Удобен для добавления новых КП в ПЗ.

![](search2.png)

### Поиск по участку через меню фильтров
В поле поиска введите номера КП через пробел.
Можно ввести как один номер КП, так и множество.
                                                                                                                                                                     
![](mass_search2.png)
