# Экспорт списка КП

В редакторе план-заданий можно выгрузить в таблицу все КП муниципального образования или целого района.

Выберите участок для выгрузки в меню фильтров. Cформируется список КП данного участка.

![](unloading1.png) 
 
Нажмите троеточие и выберите формат для экспорта КП выбранного участка.

![](unloading2.png)
