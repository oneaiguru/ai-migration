# История изменений
                                                                   
Нажмите троеточие и затем "История изменений", чтобы увидеть историю редактирования план-задания.
 
![](user_activity_pz.jpg)

Откроется окно со списком изменений.
![](user_activity_pz1.jpg)

Если строка не видна целиком, нажмите на нее и она раскроется.

![](user_activity_pz2.jpg)

Предусмотрено окно поиска.
Поиск работает по номеру КП, адресу, пользователю.

![](user_activity_3.png)

### Виды событий в истории:

- создание ПЗ
- добавление КП в ПЗ 
- удаление КП из ПЗ
- изменение перевозчика 
- перенос КП между ПЗ 
- закрытие смены водителем
- принятие смены мастером
                                                         	 
По каждому событию отображается пользователь:
- Система - для действий, которые сформировались автоматически
- ФИО пользователя - для действий, произведенных вручную
- Приложение - для действий, произведенных на стороне мобильного приложения для водителей
