# Весовое план-задание для ТС 2-го плеча.

Для ТС 2-го плеча можно создать план-задание, указав места его загрузок и разгрузок и порядок, в котором они должны выполняться. 

#### Создание весового план-задания:
- Перейдите в редактор план-заданий, выберите ТС 2-го плеча и на вкладке "Вес" добавьте места его загрузок и разгрузок в весовое план-задание, используя кнопки "Добавить загрузку", "Добавить разгрузку".

![](weight_task1.png)

В мониторинге на вкладке "Вес" и у водителя в мобильном приложении теперь отображается сформированное весовое план-задание. 

#### Весовое план-задание в Мониторинге:

На вкладке "Вес" в Мониторинге также, как и в редакторе план-заданий, можно добавить удалить загрузку/разгрузку. 

 ![](weight_task2.png)

Отчеты о загрузке/разгрузке, который формирует водитель в мобильном приложении, содержат вес с ТКО / без ТКО. При необходимости выбранные значения можно скорректировать. 

![](weight_task3.png)

Если используется прицеп, то водитель отдельно указывает вес прицепа с ТКО/без ТКО. В "Итого" записываются общий вес загрузок/разгрузок в рамках план-задания. 

![](weight_task4.png)

По умолчанию ТС 2-го плеча не вывозит отходы с КП. В настройках системы есть параметр "ТС 2-го плеча может загружаться на КП", который при необходимости можно включить. 
Если он включен, то в справочнике можно будет отметить ТС 1-м и 2-м плечом и одновременно создать и план-задание из КП, и весовое план-задание. 

Указать плечо ТС можно в справочнике ТС. [см. подробнее](/docs/disp/prepare/ts)
