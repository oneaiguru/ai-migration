# Перевозчики

В разделе Настройки на вкладке Перевозчики можно добавить и редактировать перевозчиков.

![](carrier1.png) 

Доступ к разделу Настройки имеет только пользователь с ролью ` Администратор `.

Чтобы создать нового перевозчика нажмите "Добавить", откроется меню создания перевозчика.

![](carrier2.png) 

- Автомат. генерация план-задания - ПЗ на завтра будет формироваться автоматически, если для ТС задан маршрут по умолчанию.

- Активный - перевозчик может быть активным либо архивным, когда тумблер выключен. Архивный перевозчик не доступен пользователям для работы.

- Использует гаражные номера ТС - для синхронизации ТС с gps-сервисом через гаражные номера.

Для удаления перевозчика нажмите на иконку корзины.
