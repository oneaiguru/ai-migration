# Сотрудники

## Создание пользователя

В таблицу ` Справочники - Сотрудники ` необходимо добавить -

1. Пользователей с доступом в веб-интерфейс Чистой логистики (диспетчеры, мастера участков и т.п.)
2. Водителей мусоровозов

![](employees2.png) 

Для создания нового пользователя нажмите "Добавить", откроется окно создания пользователя.
Обязательные поля у каждого пользователя - фамилия, имя, список перевозчиков и роль в системе.
 
![](employees1.png) 

Пользователь может самостоятельно сменить пароль в справочнике сотрудников.

![](12345.png) 

После ввода фамилии и имени, доступна возможность сгенерировать логин и надежный пароль. 

![](employees3.gif) 

При создании учетной записи водителю также задается логин и пароль, но для работы в мобильном приложении логин и пароль не потребуется.

![](employee-2.png)

- администратор - доступ к разделу Настройки. 

![](employee-1.png)

## Регоператор

Включенная опция "регоператор" в списке перевозчиков дает пользователю доступ ко всем перевозчикам в системе. 

![](2022-04-05-16-11-48.png)

В мониторинге: рег.оператор имеет возможность редактировать любые старые отчеты по КП. Остальные пользователи могут редактировать отчеты только за прошлый месяц до 4 числа текущего месяца.

В зависимости от организации работы в каждом регионе могут быть установлены следующие правила:

- в справочнике ТС: устанавливать машинам статусы "В работе", "На проверке" и "Отказ" имеет право только сотрудник с опцией рег.оператор.

![](2022-05-28-19-23-41.png)

- в реестре КП: создать КП может только только сотрудник с опцией рег.оператор.
