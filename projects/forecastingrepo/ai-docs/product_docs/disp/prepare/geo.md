# Гео-объекты

В разделе Настройки на вкладке Гео-объекты можно добавить и редактировать:
- гаражи
- мусорные полигоны 
- места сортировки и мусороперегрузочные станции
- КПО (Комплекс переработки отходов), или иными словами, мусороперерабатывающий завод

Доступ к разделу Настройки имеет только пользователь с ролью ` Администратор `.

![](geo1.png) 

Для создания нового гео-объекта нажмите "Добавить", откроется окно создания объекта.

![](geo-2.png) 

Для каждого объекта требуются его название, адрес и координаты. 
Внесенные объекты используются для расчетов оптимального маршрута и отображаются на карте:

![](2021-10-11-18-19-41.png)
