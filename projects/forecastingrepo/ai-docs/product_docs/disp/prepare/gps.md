# GPS-система

Мусоровозы должны быть оснащены датчиками системы GPS/Глонасс. 

В разделе Настройки на вкладке GPS-пользователи указываются учетные данные для синхронизации с gps-сервисом мусоровоза.

Доступ к разделу Настройки имеет только пользователь с ролью ` Администратор `.

![](gps1.png) 

Для синхронизации с новой gps-системой нажмите "Добавить", откроется окно для добавления gps-пользователя.

![](gps2.png) 

Необходимо указать:

- Название используемого GPS-cервиса выбирается из предложенного списка
- url адрес, а также логин и пароль для подключения
