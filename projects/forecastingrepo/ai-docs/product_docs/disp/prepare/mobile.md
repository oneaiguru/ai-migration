# Мобильные устройства

Мобильное приложение для получения отчетов от водителей мусоровозов и проведения инвентаризации работает на устройствах с Android версии 7.0 и выше. Установить приложение можно из:

- [Google Play](https://play.google.com/store/apps/details?id=com.groupstp.smartdispatch_driver)
- [RuStore](https://www.rustore.ru/catalog/app/com.groupstp.smartdispatch_driver)

Требования к устройству:
- Планшет с экраном 8”-9” или телефон с экраном от 5”
- Разрешение экрана - от 1280х800рх
- RAM от 2 Гб
- Интернет - устройство должно иметь поддержку сим-карт с 4G
- Фото - устройство должно иметь камеру
- GPS - устройство должно иметь датчик GPS/Глонасс и он постоянно включен
- Аксессуары - чехол и защитное стекло для планшета

Рекомендуемая модель (по результатам тестов и опыту эксплуатации) - Samsung Tab A7.

## Регистрация устройств

Для работы необходимо зарегистрировать устройство на сервере и либо назначить его на машину, либо отметить его для проведения инвентаризации.

1. При первом запуске приложения выберите свой регион и перевозчика из списка.
1. Зайдите в раздел ` Справочники - Устройства ` 
1. Найдите в списке ID устройства и выберите:
  - номер ТС из списка, если устройство предназначено для водителя мусоровоза
  - ` Назначить на инвентаризацию ` в меню, если устройство будет использовано для инвентаризации КП

## Отключение режима Talkback для слабовидящих
По ошибке пользователь может включить режим для слабовидящих, зажав случайно кнопки громкости. Планшет начнет озвучивать все, что видит на экране и ваши действия. 

Для отключения режима talkback выполните следующие действия на планшете:
1. На боковой стороне устройства находятся кнопки регулировки громкости.
2. Нажмите и удерживайте обе кнопки в течение трех секунд.
3. Для подтверждения действия снова нажмите и удерживайте обе кнопки громкости в течение трех секунд.
