# Подготовка к работе

Чистая логистика не требует установки на компьютер, для пользователей создается аккаунт в облачном сервисе и вся работа с ПО производится через браузер Интернета.

Для начала работы пользователь с правами администратора вносит в Чистую логистику данные по:

1. [Перевозчикам](carrier)
1. [Участкам вывоза](regions)
1. [Гео-объектам](geo) (гаражи, мусорные полигоны, МПС)
1. [Сотрудникам](employees)
1. [Транспортным средствам](ts)
1. [GPS-датчикам](gps)
1. [Мобильным устройствам](mobile)

## Реестр контейнерных площадок

Подготовка полного и корректного реестра КП может занять время, но является критичной для запуска системы в работу. Чтобы не вносить данные о каждой площадке вручную, 
можно передать в техподдержку Чистой логистики [реестр в Excel-формате](pathname:///files/template.xlsx). 

Если есть сомнения в полноте имеющегося реестра, нужно запустить процесс [инвентаризации контейнерных площадок](/docs/disp/registry/inventory). 

## Подготовка план-заданий

На первом шаге создаются [маршруты](/docs/disp/routes/) - шаблоны, содержащие список КП для каждой машины. Для ежедневного вывоза из маршрутов создаются [план-задания](/docs/disp/plan/plans) на конкретный день.

## Мобильные устройства

Приложение водителя работает с Android-планшетами и телефонами. [Подробнее о настройке устройств.](mobile)

## Поддержка

Техподдержка Чистой логистики поможет по любым вопросам работы с системой, в том числе на этапе подготовки к работе с ней. Напишите в [наш Telegram](https://t.me/clean_logistics_bot) или на почту logistics@mytko.ru.
