# Участки вывоза

**Участок вывоза** - населенный пункт или его район, в рамках которого осуществляется вывоз ТКО. Как правило, в момент начала работы с Чистой логистикой уже есть сложившаяся практика, на каких участках работает регоператор.

В разделе Настройки на вкладке Районы можно создать и редактировать районы.
Доступ к разделу Настройки имеет только пользователь с ролью ` Администратор `.


![](region1.png) 
          
Для создания нового района нажмите "Добавить", откроется меню создания района.

![](region2.png) 

- цвет заливки определяет каким цветом будет выделен участок на карте. метод задания цвета - шестнадцатеричные цвета.
- родитель - поле предусмотрено для формирования иерархии участков

![](region3.png) 

- координаты участка задаются в виде массива точек, который преобразуется в полигон.
