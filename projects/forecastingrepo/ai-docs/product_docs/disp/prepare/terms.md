# Терминология и сокращения

**КП** - Контейнерная площадка

**Контейнерная группа** - контейнеры одинакового объема на одной КП

**ТС** - Транспортное средство

**Экипаж** - Водители, назначенные на ТС

**План-задание** - Список КП для вывоза заданной ТС в заданный день

**Маршрут** - Список КП, шаблон для создания план-задания

**Инвентаризация** - Сбор начальных данных о КП

**Верификация** - Процесс изменения данных о КП

**Реестр КП** - Список всех КП регоператора
