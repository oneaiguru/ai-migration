# Транспортные средства

В таблицу ` Справочники - Транспортные средства ` добавляются мусоровозы, участвующие в вывозе ТКО и мусоровозы второго плеча. Данные по ТС включают в себя -

- Госномер
- Cтатус	
- Модель		
- Способ погрузки
- Участок вывоза
- Экипаж
- Гараж	
- График работы
- Координаты GPS
- [Маршрут по умолчанию](../routes/routes)

### Статус ТС

ТС может иметь следующие стандартные статусы:
- в работе: ТС можно использовать для вывоза мусора
- архивная: ТС перенесена в архив
- резерв: ТС на ремонте или по иным причинам не может участвовать в вывозе. 

![](ts1.png)

Можно активировать модуль контроля за ТС. 
В этом случае перевозчик имеет право только создавать новое ТС. Новое ТС будет иметь статус "На проверке".
Все последующие манипуляции с ТС выполняет рег.оператор.

При активном модуле контроля за ТС список статусов расширен:
- на проверке: для новых ТС 	
- отказ: ТС не прошла проверку регоператора
- в работе: ТС можно использовать для вывоза мусора
- архивная: ТС перенесена в архив
- резерв: ТС на ремонте или по иным причинам не может участвовать в вывозе.

![](ts2.png)

В каждом справочнике предусмотрена выгрузка. В правом верхнем углу справочника нажмите на экспорт 
для выгрузки списка ТС, сотрудников или планшетов. 

![](unloading-directory.png) 

### Плечо мусоровоза

- Мусоровозы, которые вывозят ТКО с КП отмечаются в таблице ТС атрибутом "Первое плечо".
- Мусоровозы, которые перевозят ТКО между местами перегруза/сортировки и захоронения отмечаются в таблице ТС атрибутом "Второе плечо". 
В разделе "Мониторинг",  "План-задания" ТС 2-го плеча отображается серым цветом. Для ТС 2-го плеча план-задание не является обязательным, поэтому вкладка "План-задание" по умолчанию скрыта, появляется вкладка "Вес". 

![](ts12.png)

По умолчанию ТС 2-го плеча не вывозит отходы с КП и движется только между полигонами. В настройках системы есть чек-бокс "ТС 2-го плеча может загружаться на КП", который при необходимости можно включить. Если чек-бокс включен, то в справочнике ТС можно отметить 1-м и 2-м плечом одновременно и создать план-задание из КП.
