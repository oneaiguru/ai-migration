# Свойства контейнерной площадки

![](2024-03-30-19-01-40.png)

**Код КП** - уникальный 8-значный номер КП. Назначается автоматически при создании КП, первые две цифры - код региона.

**Дата создания** - дата, когда КП была создана в системе. Может быть отредактирована вручную.

**Район** - район или МО, в котором расположена КП. Выбирается автоматически по координатам КП и не может быть изменен вручную.

**Адрес КП** указывается в текстовом формате. Рекомендуется не указывать в нем название населенного пункта для удобства работы со списком адресов в дальнейшем.

**Координаты КП** - обязательное поле, могут быть введены вручную или кликом в точку на карте.

**Категория**, **тип КП** и **доступ по габаритам** - справочные поля.

**Расчет тарифа** - Норматив или Факт. Если используется интеграция с Личным кабинетом, может быть выключен для изменения в Чистой логистике.

**Доступ по дням** позволяет указать дни недели, когда площадка доступна к вывозу. Если график вывоза КП выпадет на недоступный день, вывоз будет автоматически перенесен на ближайший доступный.

**Доступ в выходные** - аналогичное по смыслу свойство. Если оно выключено, это означает недоступность площадки в выходные и праздничные дни. Вывоз с таких дней будет автоматически перенесен на ближайший доступный.

**Запрет фото** - если включено, водитель не сможет сделать фотографии при вывозе площадки (используется для военных частей, аэропортов и подобных объектов, где запрещено делать фотографии).

**Режимный объект** - означает, что водителю необходимо подготовить пропуск для заезда на объект.

**Площадка под КГО** - обозначает наличие или отсутствие на площадке подготовленного отсека для КГО.

**Заполненность** - справочная информация на основе статистики вывоза площадки.

**Фото КП** - образцовые фотографии КП, помогающие сориентироваться водителю при поиске КП. Список фото создается автоматически на основе отчетов водителей, но может быть скорректирован вручную. Для этого нажмите на ссылку "Редактировать" и отметьте нужные фотографии (не более 10 штук).
