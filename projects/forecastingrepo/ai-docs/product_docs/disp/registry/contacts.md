# Контакт для уведомлений

Если включен сервис создания писем контранентам, в свойствах контейнерной площадки появится блок "Контакт для уведомлений":

![](2024-04-29-11-57-45.png)

Указанные в нем данные будут использоваться по умолчанию при отправке писем по данной КП.
