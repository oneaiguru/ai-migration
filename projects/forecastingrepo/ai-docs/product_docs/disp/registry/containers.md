# Свойства контейнерной группы

**Контейнерной группой** называется группа из нескольких контейнеров одинакового объема на одной контейнерной площадке. У каждой группы может быть свой график вывоза, свой маршрут и даже отдельный перевозчик. 

![](2024-04-29-11-07-11.png)

## Свойства контейнеров

![](2024-04-29-11-14-27.png)

Для контейнерной группы указываются -

- число контейнеров
- тип отходов - ТКО, КГО, РСО, мешки или прочие отходы
- для РСО дополнительно указываются материал и цвет контейнера
- перевозчик указывается, только если разные контейнеры на площадке вывозятся разными перевозчиками
- место разгрузки - рекомендуемое место разгрузки для этой группы, заданное территориальной схемой обращения с отходами
