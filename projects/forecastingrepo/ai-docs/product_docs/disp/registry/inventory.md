# Инвентаризация КП
                
Инвентаризация в целом состоит из двух этапов:

1. Сбор данных о КП
2. Создание из них реестра КП

Этап сбора данных в Чистой логистике состоит из нескольких шагов:

1. [Подготовка мобильных устройств](../prepare/mobile)
1. Добавление пользователей, проводящих инвентаризацию
1. Подготовка планов осмотра
1. Обход территории с мобильным приложением
1. Верификация полученных из приложения отчетов
1. Передача данных в Личный кабинет
