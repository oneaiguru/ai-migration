# Реестр КП

Реестр КП содержит полный список контейнерных площадок в зоне ответственности регионального оператора. Добавление площадки в реестр может быть сделано вручную либо через процессы верификации и инвентаризации КП.

Созданную площадку нельзя удалить из реестра насовсем. Площадки, по которым не осуществляется вывоз,
убираются в архив кнопкой "В архив" и затем могут быть найдены через фильтр В работе / В архиве.

### Редактирование КП

Редактирование КП доступно только пользователям с ролью "Редактор реестра". Редактировать можно одну или несколько КП сразу. Для редактирования нескольких КП выберите их в списке зажав Ctrl или Shift, и нажмите на кнопку "Редактировать КП". 

![](registry6.png)

Выберите свойства КП и укажите новые значения. Новые свойства будут применены для всех выбранных КП.

![](registry7.gif)

### Интеграция с Личным кабинетом myTKO

В зависимости от региона внедрения реестр КП может быть реализован с интеграцией с модулем Личный кабинет myTKO. В этом случае изменение свойств КП в интерфейсе Чистой логистики синхронизируется с реестром КП в Личном кабинете, и наоборот. При этом редактирование КП с тарифом "Факт" может не отразиться сразу же в реестре, т.к. потребует согласования регоператором в интерфейсе Личного кабинета.
