# Маршруты

Этот диалог позволяет указать, в каких маршрутах находится контейнерная группа. Если маршрут только один, редактирование графика маршрута отключено - предполагается, что он не используется.

![](2024-04-29-11-33-13.png)

Контейнерная группа может находиться в нескольких маршрутах. При этом должен быть указан график маршрута - дни недели, по которым контейнерная группа будет вывозиться в рамках того или иного маршрута. Для исключения ошибок нужно, чтобы каждый день был отмечен только один раз.

![](2024-04-29-11-35-15.png)

При попытке оставить какой-то день пустым, без выбранного маршрута, система выдаст ошибку и не даст сохранить такой график. При выборе нескольких маршрутов в один день ошибки не будет, такие ситуации иногда возникают, но ответственность на правильность такого решения лежит на пользователе.
