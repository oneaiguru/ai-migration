# График вывоза

Для контейнерной группы необходимо указать один из возможных вариантов графика вывоза:

### Дни недели

![](2024-04-29-11-21-11.png)

Этот вид графика также используется для ежедневных площадок, так как показано на скриншоте.

### По дате

Для вывоза по указанным числам каждого месяца.

![](2024-04-29-11-23-27.png)

### Повторный

Для указания выбранных дней недели в течение месяца. Рекомендуется также для графиков вида "последний понедельник месяца".

![](2024-04-29-11-24-35.png)

### Интервальный

Для графиков вида "по четным/нечетным дням", "через день", "каждые три дня" и т.п.

![](2024-04-29-11-26-22.png)

### Календарный

Используется, когда в договоре с контрагентом указываются конкретные календарные даты вывоза.

![](2024-04-29-11-27-55.png)

### По заявке

Используется, когда график вывоза заранее неизвестен и вывоз осуществляется при получении заявки от контрагента.

### Субботник и Под загрузку

Аналоги графика "по заявке", которые подразумевают установку контейнера на площадку непосредственно перед её вывозом.
