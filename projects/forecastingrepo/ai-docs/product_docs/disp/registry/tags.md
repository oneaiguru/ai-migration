# Метки

Метки позволяют -

- задать произвольные отметки для КП и затем использовать их в фильтрах, редакторе и мониторинге
- отметить особо важные КП
- объединить КП в группы и таким образом изменить интерфейс приложения водителя, который вывозит эти КП

В реестре КП, редакторах и мониторинге КП с метками выделяются цветным индикатором, при наведении курсора на него показывается название метки:

## Редактирование меток

Чтобы выбрать метку из созданных ранее, выберите ее из списка меток:

![](2024-04-29-12-09-14.png)

Это также можно сделать в массовом редактировании КП:

![](2024-04-29-12-09-59.png)

Чтобы добавить новую метку или изменить созданную ранее, выберите "Редактировать метки..." в этом списке.

![](2024-04-29-12-11-16.png)

Для каждой метки можно указать ее название, цвет и, если нужно, значение. Значений существует три:

### 1. Особо важная КП

Отмеченные этой меткой КП в ПЗ мониторинга, если произошел невывоз по такой КП, выделяются красным.

![](2024-04-29-12-13-01.png)

При приеме смены, если в ней остались невывозы по особо важным КП, выдается сообщение "Внимание! В план-задании есть особо важные КП, по которым был зафиксирован невывоз."

### 2. Группа КП

Метка "группа КП" позволяет упростить водителю вывоз в случаях, когда например подъезды в многоквартирном доме представлены в реестре как отдельные КП. В приложении водителя все КП одной группы схлапываются в одну строку и отмечаются двойной иконкой:

![](2024-04-29-12-16-36.png)

В окне отчета с КП водитель видит все КП данной группы и может отправить отчет по ним одним нажатием кнопки "Отправить отчет".

![](2024-04-29-12-17-34.png)

### 3. Группа ИЖС

Метка по своим свойствам аналогична метке "Группа КП", но у отмеченных ей КП в приложении водителя снимается требование обязательных фотографий. Для КП с меткой "Группа ИЖС" водитель может отправить отчет без фотофиксации сразу по всем КП в группе.
