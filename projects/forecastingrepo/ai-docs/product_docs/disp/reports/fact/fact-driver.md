# Отчет по водителям

Отчет по водителям находится  в разделе Отчеты - Отчет по факту.

Отчет формируется по водителям перевозчика и отображает процент выполнения в разрезе по: 
- план-заданию
- контейнерам
- объему ТКО

Таже в отчете отображается количество и вес разгрузок на полигонах.

![](factDriver1.png)

Отчет можно экспортировать в эксель, пдф, гугл и яндекс таблицы.
