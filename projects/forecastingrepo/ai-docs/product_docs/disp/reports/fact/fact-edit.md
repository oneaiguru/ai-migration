# Отчет Корректировка

Отчет по корректировке находится  в разделе Отчеты - Отчет по факту.

Отчет необходим для корректировки объемов вывоза по КП и доступен пользователю с правами рег.оператора.

![](factEdit1.png)
               
Доступные для корректировки поля: факт, подбор.  
Отчет можно экспортировать в эксель, пдф, гугл и яндекс таблицы.
