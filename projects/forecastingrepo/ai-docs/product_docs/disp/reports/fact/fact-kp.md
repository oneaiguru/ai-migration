# Отчет по площадке

Отчет по площадке находится в разделе Отчеты - Отчет по факту.

В отчете по площадке формируется факт вывоза КП за период.  
Отчет включает фотографии "до" и "после" вывоза и может быть выгружен в pdf.

В поле поиска введите номера КП через пробел.
Отчет сформируется для всех КП из списка.
