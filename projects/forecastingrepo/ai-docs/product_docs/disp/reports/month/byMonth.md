# Отчет по месяцам

Отчет по месяцам находится в разделе Отчеты - По месяцам.

Отчет представлен в двух форматах:

- по участкам - объем собранного мусора, общий вес разгрузок и суммарный пробег ТС перевозчика за месяц.

![](month1.png)

- по ТС - объем собранного мусора, общий вес разгрузок и пробег за месяц по каждому ТС перевозчика.

![](month2.png)
