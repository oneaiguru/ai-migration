# Среднее плечо

Отчет по среднему плечу находится  в разделе Отчеты - Транспорт.

В отчете отражаются средний пробег всех ТС участка 1 и 2 плеча до объекта разгрузки и средневзвешенное значение перевозимого ТКО на объекты разгрузки:
- участок - ТС участка, который учитывается для подсчета
- плечо - тип плеча ТС, которое учитывается для подсчета
- Объект 1 плеча транспортирования - название объекта, который учитывается для подсчета
- Объект 2 плеча транспортирования - название объекта, который учитывается для подсчета

Пробег за 1 рейс
- средний пробег, км - среднее арифметическое пробегов всех ТС за период/число рейсов
- средневзвешенное, тонн - сумма всех (пробег за рейс)*(вес за рейс) за период/сумма всех весов
- средневзвешенное, м3 - сумма всех (пробег за рейс)*(объем за рейс по план-заданию) за период/сумма всех объемов

Данные терсхемы и отклонение
- среднее из терсхемы, км - устанавливается в настройках системы
- отклонение, км - средний пробег за 1 рейс - среднее из терсхемы 
- средневзвешенное, км/ тонн - устанавливается в настройках системы
- отклонение, км/тонн - средневзвешнное за 1 рейс - средневзешенное из терсхемы
