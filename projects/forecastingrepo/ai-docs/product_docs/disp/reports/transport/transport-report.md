# Время заезда/выезда

Отчет по времени заезда/выезда находится  в разделе Отчеты - Транспорт.

В отчете отражается данные по ТС перевозчика:
- время выхода на линию
- время заезда в гараж
- рабочее время
- пробег, км
- расход топлива, л
- КТУ (коэффициент трудового участия) водителя

![](inOut1.png)

Для правильного расчета в справочнике ТС должно быть заполнено поле  "гараж" для каждого ТС.

Отчет можно экспортировать в эксель, пдф, гугл и яндекс таблицы.
