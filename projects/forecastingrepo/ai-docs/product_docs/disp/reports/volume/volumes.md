# Отчет по объемам

Отчет по объемам находится в разделе Отчеты - По объемам.

Для работы с отчетом установите необходимые фильтры: перевозчик, район вывоза, период отчета.  
Отчет по объему вывезенных контейнеров считается в контейнерах или в кубометрах.

В отчете отражается объем ТКО, собранный по каждой КП участка, в разрезе за каждый день.
Если водитель в приложении указал проблему на КП, то она отобразится в столбце "Проблема".
