# Отчет по весу
Отчет по весу находится в разделе Отчеты - По весу.

Отчет отражает загрузки и разгрузки ТС на местах хранения ТКО и включает название полигона, время и вес загрузки либо разгрузки. 
Для работы с отчетом можно установить фильтры:
- выбор места хранения ТКО - отчет сформируется по ТС, которые загружались либо разгружались на данном месте хранения ТКО.
- выбор плеча - отчет сфомируется по ТС определенного плеча.

У ТС 1 плеча, в отличие от ТС 2 плеча, есть только разгрузки и столбец с загрузками у ТС 1 плеча будет пустой.

![](weight2.png)

Отчет может быть сформирован в двух форматах
- краткий отчет по весу
- детальный отчет по весу

В детальный отчет по весу добавляется вес ТС с ТКО и без ТКО, а также объем кузова и коэффициент сжатия ТС.

![](weight3.png)

Отчет можно экспортировать в эксель, пдф, гугл и яндекс таблицы.
