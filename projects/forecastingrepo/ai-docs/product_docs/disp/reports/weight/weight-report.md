# Баланс масс

На каждый день для места разгрузки будет посчитан остаток. Для расчета первый день берется из настроек, следующий считаем как "поступление - убытие + остаток прошлого дня".

Для МПС, МСС и полигонов укажите параметры: остаток в тоннах и дату отсчета. Параметры задаются в настройках гео-объектов и являются начальными для отчета. Отчет формируется на каждый день отдельно для каждого места разгрузки. 

![](2911-1.jpg)
       
- Поступление - сумма всех весовых отчетов о разгрузке 
- Убытие - сумма всех весовых отчетов о загрузке 
- Баланс масс = поступление - убытие
- Остаток = поступление - убытие + остаток прошлого дня
- Число рейсов - число весовых отчетов о разгрузке
