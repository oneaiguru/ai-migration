# Удаление КП из маршрута

Чтобы удалить контейнерную группу из маршрута выберите одну или несколько (зажать shift или ctrl) контейнерных групп. 
Нажмите "удалить".

![](delete-kg1.png) 

#### При изменении маршрута, созданное на его основе план-задание также изменится. Синхронизация работает только для будущих дат. Без изменения остаются план-задания на сегодня и прошлые даты.
