# Прогноз загрузки ТС

При разработке маршрута важно заранее понять, к каким характеристикам созданных из него план-заданий он в итоге приведет. 
Для их оценки в редактор маршрутов встроен прогноз загрузки ТС. Чтобы открыть его, нажмите на иконку прогноза в редакторе. Таких иконок две - одна показывает прогноз по текущему выбранному маршруту:

![](2024-04-29-18-02-25.png)

Вторая откроет прогноз для всех маршрутов участка.

![](2024-04-29-18-03-04.png)

Прогноз представляет из себя график, на котором показан суммарный объем вывоза на каждый день, если план-задание будет создано из заданного маршрута. Прогноз строится на 30 дней вперед относительно сегодняшней даты.

![](2024-04-29-18-03-52.png)

Параметр "По номинальному объему / С учетом заполненности КП" указывает, нужно ли в прогнозе объема учитывать [коэффициент заполненности КП](../control/report-kp) или нет.

Наведите курсор на столбец, чтобы увидеть данные прогноза на этот день.

![](2024-04-29-18-08-55.png)

Если у ТС, с которой связан маршрут, указаны объем кузова и коэффициент сжатия, график прогноза будет поделен на рейсы.

![](2024-04-29-18-10-29.png)

Сегмент красного цвета означает, что по прогнозу машина последним рейсом соберет незначительное количество ТКО (менее 10% от объема кузова). Сегмент желтого цвета обозначает менее 50% загрузки кузова в последнем рейсе.

Синим цветом на графике показаны фактические данные по вывозу для выбранной ТС, чтобы их можно было сравнить с прогнозными:
