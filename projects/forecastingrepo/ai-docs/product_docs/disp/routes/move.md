# Перенос КП между маршрутами
 
Функция переноса КП используется, как инструмент редактирования созданных маршрутов. 

Выберите одну или несколько контейнерных групп и нажмите "Перенести".

![](move1.png)

В окне переноса выберите необходимый участок, маршрут и нажмите "Перенести". 

![](move2.png) 

#### При изменении маршрута, созданное на его основе план-задание также изменится. Синхронизация работает только для будущих дат. Без изменения остаются план-задания на сегодня и прошлые даты.
