# Создание маршрутов

Для корректного создания план-заданий нужно, чтобы все КП с установленным графиком вывоза были привязаны к маршрутам. Исключение можно сделать для КП с графиком "По заявке".

1. В редакторе маршрутов выберите "Новый маршрут"

![](2021-10-13-19-22-54.png)

Укажите произвольное название для нового маршрута. Пользователь-регоператор должен привязать маршрут к перевозчику и участку. Пользователь-перевозчик - только к участку.

:::info ВАЖНО

Для создания маршрутов в заданном участке необходимо, чтобы у перевозчика в справочнике ТС одна или более машин принадлежали этому участку. Создать машруты в участке, где нет ТС, не получится.

:::


![](2021-10-13-19-26-51.png)

2. Добавьте в список нужные КП. Это можно сделать:

- на карте. Для этого выберите на карте режим рисования, обведите нужные точки и нажмите "Добавить в маршрут".

![](2021-10-13-19-30-46.png)

![](2021-10-13-19-32-19.png)

- поиском по адресу или коду КП

![](2021-10-13-19-33-56.png)

![](2021-10-13-19-36-46.png)

Если на КП находятся контейнеры разного объема и их нужно вывозить разнами машинами, можно разнести контейнерные группы КП по разным маршрутам.

3. Чтобы включить автоматическое создание план-заданий на основе маршрута, выберите ТС (раздел Справочники - Транспортные средства) и укажите ее маршрут по умолчанию.

Маршрут может остаться не закрепленным ни за одной машиной. в этом случае создать план-задание на его основе можно вручную в [редакторе план-заданий](../plan/plans).

![](2021-10-13-19-46-12.png)

### Обновление план-заданий

 При изменении маршрута также изменятся план-задания, созданные на его основе на будущие даты. План-задания на сегодня и прошлые даты останутся без изменений.
