# Маршруты
                   
Все КП заданного участка разбиваются на списки, называемые шаблонами маршрутов или просто **маршрутами**. Маршрут это заготовка, список площадок, которые планируется назначить к вывозу на одну машину. План-задание на нужный день создается из маршрута.


Для каждой машины можно указать её маршрут по умолчанию, тогда план-задание будет формироваться автоматически на основе маршрута:

1. Создайте маршрут и привяжите его в справочнике ТС к машине
1. План-задание будет формироваться на завтра автоматически. В ПЗ попадут КП из маршрута, у которых график вывоза попадает в данный день
