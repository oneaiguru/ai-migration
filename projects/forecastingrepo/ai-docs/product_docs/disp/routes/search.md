# Поиск

В редакторе маршрутов поиск разделен на два вида. 
Глобальный поиск и поиск по участку через меню фильтров. 

В редакторе маршрутов две таблицы - с текущим маршрутом 
и контейнерными площадками выбранного участка. 
В окне глобального поиска можно выбрать, где он будет происходить.

![](search1.gif)

### Глобальный поиск по участку 

Используется, чтобы найти КП в таблице участка. Поиск по участку удобен для добавления новых КП в маршрут.

![](search1.png)


### Глобальный поиск по маршруту 

Кроме участка, откроется маршрут, в котором находится КП.

![](search2.png)

### Поиск по участку через меню фильтров
В поле поиска по участку введите номера КП через пробел.
Можно ввести, как один номер КП, так и множество.
