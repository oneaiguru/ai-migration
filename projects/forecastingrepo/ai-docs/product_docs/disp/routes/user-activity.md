# Логирование действий пользователя

Инструмент логирования действий пользователя полезен:
- для непрерывного контроля за изменениями 
- для продуктивной работы нескольких диспетчеров над одной задачей

### История изменения маршрута
                                                                   
Нажмите троеточие и затем "История изменений", чтобы увидеть историю редактирования маршрута.
 

![](user_activity_route.png)

Откроется окно со списком изменений.
![](user_activity_route1.png)

Предусмотрено окно поиска.
Поиск работает по номеру КП, адресу, пользователю.

![](user_activity_route2.png)

### Виды событий в истории:
- создание маршрута
- изменение названия маршрута
- добавление КП в машрут
- удаление КП из маршрута
- изменение перевозчика 
- перенос КП между маршрутами
