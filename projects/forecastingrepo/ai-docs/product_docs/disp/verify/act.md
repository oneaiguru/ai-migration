# Актирование мастером

Выберите заявку в списке и нажмите на кнопку "Назначить на мастера". В открывшемся диалоге выберите одного или нескольких мастеров участка.

Для актирования КП мастер должен запустить мобильное приложение "Чистая логистика" в режиме инвентаризации. После введения логина и пароля он увидит список всех назначенных на него верификаций.

Мастер должен в приложении сделать фото подъездных путей, общего вида площадки, детальные фото контейнеров и, если были указаны другие разделы акта, заполнить их.
