# Evaluation Delivery Pack — What to ZIP for Coordinator Review

## Purpose
Provide a minimal, deterministic bundle so the coordinator can:
- Re-run unit/CI gates quickly (no raw data or secrets),
- Inspect accuracy numbers (WAPE/SMAPE/MAE) across 3 cutoffs,
- See simple visuals to catch anomalies,
- Read provisional T₁/T₂ (before PR‑15 rolling aggregator).

## INCLUDE (exact paths)
- Code & tests:
  - scripts/, src/, specs/, tests/, .tools/, .github/
  - scenarios/
  - requirements*.txt, README.md, AGENTS.md
- System & research docs:
  - docs/System/, docs/architecture/, docs/SOP/, docs/adr/, docs/prd/
  - docs/data/, docs/eval/, docs/research/, docs/Tasks/
  - docs/contract/thresholds_provisional.md  (generated)
- Evaluation data & reports **only**:
  - data/eval/actuals/  (tiny CSVs used by backtests)
  - reports/backtest_2024-03-31/{scoreboard_*.csv,SUMMARY.md}
  - reports/backtest_2024-06-30/{scoreboard_*.csv,SUMMARY.md}
  - reports/backtest_2024-09-30/{scoreboard_*.csv,SUMMARY.md}
  - reports/backtest_consolidated_*/  (from concat_scoreboards.py; histograms + SUMMARY.md)
  - reports/quicklook_demo/  (if generated by scripts/quicklook_report.py)

## EXCLUDE
- .git/, __pycache__/, .pytest_cache/, .coverage, coverage.xml
- deliveries/, delivery/, forecasts/
- data/raw/**, *.xlsx
- spec_registry/spec_index.yml
- any secrets (.env, ~/.cdsapirc)

## How to produce (agent)
1) Ensure backtests exist (three cutoffs): see docs/Tasks/ACCURACY_LOCK_RUNS.md.
2) Aggregate & plot:
```
python scripts/eval/concat_scoreboards.py \
  --inputs reports/backtest_2024-03-31 reports/backtest_2024-06-30 reports/backtest_2024-09-30 \
  --outdir reports/backtest_consolidated_auto
```
This writes:
- reports/backtest_consolidated_auto/scoreboard_consolidated.csv
- reports/backtest_consolidated_auto/SUMMARY.md (p50/p70/p90; T₁/T₂ provisional)
- reports/backtest_consolidated_auto/hist_region.png, hist_district_micro.png
- docs/contract/thresholds_provisional.md
3) Package ZIP:
```
bash scripts/make_eval_bundle.sh
```
→ ~/Downloads/eval_bundle_YYYYMMDD_HHMM.zip

## Sign‑off gate (coordinator)
- Create venv, `pip install -r requirements-dev.txt`
- Run:
```
pytest -q --cov=scripts --cov-report=term-missing
python .tools/spec_sync.py
python .tools/docs_check.py
```
- Open `reports/backtest_consolidated_*/SUMMARY.md` and `docs/contract/thresholds_provisional.md`.
