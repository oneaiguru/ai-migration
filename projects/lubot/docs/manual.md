 Руководство Lubot

1. Начало: Отправьте `/start` для регистрации.

2. Управление партнерами:
   - `/p`: Откройте меню управления партнерами, где вы можете добавить нового партнера или удалить выбранного партнера с помощью кнопок.
   - `/r <новое_имя>`: Переименуйте выбранного партнера.

3. Советы: Общайтесь с ботом, чтобы получить советы по отношениям. Советы будут учитывать контекст выбранного партнера.

4. Поддержка: Используйте `/support` для помощи.

5. Реферальная программа:
   - `/referral`: Получите вашу реферальную ссылку для приглашения друзей.
   - `/referral_status`: Проверьте ваш баланс бонусов и статус реферальной программы.

6. Ваши отзывы напрямую влияют на улучшение продукта.

7. Получайте бонусы для более продолжительных сессий, баланс виден через `/referral_status`.

 Как работают функции партнеров:

- Использование команды `/p`: Откройте меню управления партнерами. В этом меню вы можете:
  - Добавить нового партнера: Нажмите кнопку "Добавить партнера" и введите имя нового партнера.
  - Удалить выбранного партнера: Если у вас есть выбранный партнер, нажмите кнопку "Удалить партнера", чтобы удалить его.

- Переименование партнера: Используйте команду `/r`. Например, `/r Алиса` переименует выбранного партнера в "Алиса".

Помните, что контекст выбранного партнера влияет на советы, которые вы получаете, делая их более персонализированными и релевантными для ваших текущих отношений.
