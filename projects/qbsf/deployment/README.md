# Salesforce-QuickBooks Integration

This package contains the integration middleware between Salesforce and QuickBooks Online.

## System Requirements

- Node.js v20+ (earlier versions have reached end-of-life)
- NPM v7+ (usually included with Node.js)
- Access to Salesforce and QuickBooks APIs

## Quick Start

1. Run the setup script:
   ```
   cd scripts
   ./sf-qb-setup-script.sh
   ```

2. Follow the implementation guide in the `docs` directory for complete instructions.

## Documentation

The `docs` directory contains comprehensive guides:

- implementation-guide.md - Complete implementation workflow
- middleware-setup.md - Setting up the middleware server
- sf-setup-guide.md - Salesforce configuration

- auth-config-guide.md - Authentication setup
