## Jury Demo Notes (Forecasting)

- **Data cutoff**: show “Дата среза данных” to explain what the system knows at forecast time.
- **Fact vs forecast**: blue = факт, green = прогноз, red dots = вывозы.
- **CSV export**: provide an Excel-ready CSV for the selected site/date range.

## Animation & Transitions

- Tab switching: Instant (no animation)
- Table sorting: Smooth reorder (200ms)
- Chart rendering: Fade in (300ms)
- Hover effects: Color transition (150ms)

All transitions use CSS `transition` property for performance.
